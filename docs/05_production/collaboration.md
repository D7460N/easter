# Cross-Team Collaboration

## Purpose

To ensure all creative ministries move together in harmony, supporting one message and schedule.

### Key Collaborations

| Team                     | Coordination Focus                                     |
| ------------------------ | ------------------------------------------------------ |
| **Worship Team**         | Song keys, transitions, mic sharing, rehearsal order   |
| **Tech Team**            | Cue timing, lighting levels, audio balance             |
| **Pastoral Staff**       | Theological review, messaging alignment                |
| **Hospitality & Ushers** | Audience flow, safety, communication during live shows |

### Communication Rhythm

- Weekly coordination meeting (30 min max).
- Shared rehearsal calendar online.
- One communication channel for all updates.
